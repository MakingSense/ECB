== Description ==
A light plugin that adds a column and a filter parameter to page listing in the WordPress admin area. Allowing user to filter page results by page templates.

This is particularly useful for sites that have many pages using a variety of templates. 

== Installation ==
1. Upload \"wp-admin-filter-pages-by-page-template.php\" to the \"/wp-content/plugins/\" directory.
2. Activate the plugin through the \"Plugins\" menu in WordPress.
